# Pharma Sales AI-CRM

## Project Structure
- **/aivoa-crm**: React Frontend (UI & State Management)
- **/aivoa-backend**: FastAPI & LangGraph (AI Logic & Database)

## Tech Stack
- **Frontend**: React, CSS, Node.js
- **Backend**: FastAPI, Python
- **AI**: LangGraph (Stateful Agents), OpenAI/Anthropic API
- **Database**: SQLite3

## Core Features Demonstrated
1. **Extraction**: Unstructured text to JSON.
2. **Interactive Editing**: Chat-based field updates.
3. **Sentiment Analysis**: Mood detection.
4. **Follow-up Generation**: Automated task creation.
5. **Persistence**: SQL Database storage.

## How to Run
1. Start Backend: `uvicorn main:app --reload` 
2. Start Frontend: `npm start`
